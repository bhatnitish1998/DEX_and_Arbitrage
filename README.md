# P2P Cryptocurrency Network

## Introduction
This project is an introduction to blockchain and cryptocurrencies, on building a DEX using solidity.  

## Contracts
├── contracts/  
	└── Arbitrage.sol/    
	└── DEX.sol/   
	└── LPToken.sol/    
	└── Token.sol/    

## Scripts to run experiments
├── scripts/  
	└── simulate_arbitrage.js/    
	└── simulate_dex.js/     


## Changing configuration
Modify the `thresh` variable in `scripts/simulate_arbitrage.js` to run for different threshold values.
